net.java.truevfs.access

Interface TRex

  • All Known Implementing Classes:
    TFile, TPath

    @Immutable
public interface TRex
    Defines common properties and operations of TFile and TPath. This interface is of little practical use and solely exists for providing a common abstraction layer with a common Javadoc. Application developers should not use it - hence the silly name.
    Author:
    Christian Schlichtherle
