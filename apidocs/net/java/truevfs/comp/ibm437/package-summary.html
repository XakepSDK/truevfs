Package net.java.truevfs.comp.ibm437 Description

Provides the IBM437 character set, which was implicitly assumed to be used for ZIP files before version 6-3-0 of PKWARE's ZIP File Format Specification defined it explicitly and added UTF-8 as an option. If the JRE provides its own implementation, this package will not be used and may be safely removed from the class path.
