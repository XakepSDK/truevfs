    @Immutable
public class CheckedOdfDriver
extends OdfDriver
    An archive driver for ODF files which checks the CRC32 value for all ZIP entries in input archives. The additional CRC32 computation makes this class slower than its super class.

    If there is a mismatch of the CRC32 values for a ZIP entry in an input archive, the InputStream.close() method of the corresponding stream for the archive entry will throw a Crc32Exception. Other than this, the archive entry will be processed normally. So if just the CRC32 value for the entry in the archive file has been modified, you can still read its entire contents.

    Subclasses must be thread-safe and should be immutable!
