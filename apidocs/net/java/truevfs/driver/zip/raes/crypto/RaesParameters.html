net.java.truevfs.driver.zip.raes.crypto

Interface RaesParameters

  • All Known Subinterfaces:
    RaesParametersProvider, Type0RaesParameters
    All Known Implementing Classes:
    KeyManagerRaesParameters

    public interface RaesParameters
    A marker interface for RAES parameters. RAES files feature different types to model extensibility. Each type determines the algorithms and parameter types used to encrypt and decrypt the pay load data in the RAES file. Hence, for each type a separate parameter interface is used which extends this marker interface.

    Implementations do not need to be safe for multi-threading.
