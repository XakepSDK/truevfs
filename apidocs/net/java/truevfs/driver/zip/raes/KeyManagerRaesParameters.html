Class KeyManagerRaesParameters

  • java.lang.Object
    • net.java.truevfs.driver.zip.raes.KeyManagerRaesParameters

    RaesParameters, RaesParametersProvider

    @ThreadSafe
public class KeyManagerRaesParameters
extends Object
implements RaesParametersProvider
    An adapter which provides RaesParameters by using a KeyManager for AesPbeParameters.

    The current implementation supports only Type0RaesParameters.
