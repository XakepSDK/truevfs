      • newModel

        FsModel newModel(Context context,
               FsMountPoint mountPoint,
               @CheckForNull
               FsModel parent)
        Returns a new file system model for the given mount point. This is a pure function without side effects.

        When called, you may assert the following precondition: 

        assert null == parent
         ? null == mountPoint.getParent()
         : parent.getMountPoint().equals(mountPoint.getParent());
        Parameters:
        context - the calling context.
        mountPoint - the mount point of the file system.
        parent - the nullable parent file system model.
        Returns:
        A new file system model for the given mount point.
