Class FsScheme

    @Immutable
public final class FsScheme
extends Object
implements Serializable, Comparable<FsScheme>
    Addresses a file system scheme. This is simply a URI scheme according to the syntax constraints defined in RFC 2396: Uniform Resource Identifiers (URI): Generic Syntax.

    Serialization

    This class supports serialization with both ObjectOutputStream and XMLEncoder.
