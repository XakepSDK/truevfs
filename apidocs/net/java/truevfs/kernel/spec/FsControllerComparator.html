Class FsControllerComparator

  • java.lang.Object
    • net.java.truevfs.kernel.spec.FsControllerComparator
  • All Implemented Interfaces:
    Serializable, Comparator<FsController>

    public class FsControllerComparator
extends Object
implements Comparator<FsController>, Serializable
    Compares file system controllers in reverse order of the hierarchical URI of the mount point of their file system model. When applied to a list of file system controllers for sync()ing, this ordering ensures that any archive file system gets sync()ed before their parent file system so that all file systems reflect all changes once the list has been completely processed.
