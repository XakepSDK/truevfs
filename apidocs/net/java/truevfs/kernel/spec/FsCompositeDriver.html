Interface FsCompositeDriver

  • All Superinterfaces:
    FsController.Factory<FsManager>, FsModel.Factory<FsManager>
    All Known Implementing Classes:
    FsAbstractCompositeDriver, FsSimpleCompositeDriver, InstrumentingCompositeDriver, TArchiveDetector

    @ImplementationsShouldExtend(value=FsAbstractCompositeDriver.class)
public interface FsCompositeDriver
extends FsModel.Factory<FsManager>, FsController.Factory<FsManager>
    Queries the scheme of the mount point of the given file system model in order to lookup the appropriate file system driver which is then used to create the requested thread-safe file system controller.

    Implementations should be immutable.

    Author:
    Christian Schlichtherle

        When called, you may assert the following precondition:

         ? null == model.getParent()
