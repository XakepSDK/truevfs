Class FsCovariantNode<E extends FsArchiveEntry>

  • java.lang.Object
    • net.java.truecommons.shed.UniqueObject

  • Type Parameters:
    E - the type of the mapped archive entries.
    All Implemented Interfaces:
    Cloneable, net.java.truecommons.cio.Entry, FsNode

    @NotThreadSafe
public final class FsCovariantNode<E extends FsArchiveEntry>
extends FsAbstractNode
implements Cloneable
    A covariant file system node maintains a map of archive entries and uses its key property to determine the archive entry in the map to which it forwards calls to getEntry(), Entry.getSize(Size), Entry.getTime(Access) et al.

    Christian Schlichtherle
