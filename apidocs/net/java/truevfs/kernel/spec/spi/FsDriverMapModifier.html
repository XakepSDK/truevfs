    @ServiceSpecification
public abstract class FsDriverMapModifier
extends net.java.truecommons.services.LocatableModifier<Map<FsScheme,FsDriver>>
    An abstract service for modifying maps of file system schemes to file system drivers. Modifier services are subject to service location by the FsDriverMapLocator.SINGLETON.

    If multiple modifier services are locatable on the class path at run time, they are applied in ascending order of their priority so that the result of the modifier service with the greatest number becomes the result of the entire modifier chain.
