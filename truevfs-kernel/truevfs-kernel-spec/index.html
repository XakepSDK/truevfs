About

Specifies the API for accessing the federated virtual file system space. Most noteably, the specification contains a service provider API for a singleton file system manager, an I/O buffer pool and a file system driver map.

Disclaimer

The TrueVFS Kernel is a field of constant research, so backwards incompatible changes to its API may appear anytime in order to improve it. An application developer should never use the API of this module directly, but instead use the API of the client facade module TrueVFS Access. The API of this module provides all of the functionality of the TrueVFS Kernel and more with ease and comfort.
